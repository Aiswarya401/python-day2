# Python Day 2 – Smart Password Tool

## Overview

This project is a CLI-based Password Analyzer and Generator built using Python.

It helps users check password strength and generate secure passwords with real-time analysis and file saving functionality.

---

## Features

* 🔍 Password strength checker
* 📊 Detailed analysis (length, uppercase, digits, symbols)
* 🔐 Random secure password generator
* 💾 Save generated passwords to file
* ⚡ Menu-driven CLI interface

---

## Concepts Used

* Functions
* Strings
* Lists & Conditions
* File Handling
* Python Modules (`random`, `string`)

---

## How to Run

```bash
python password_tool.py
```
